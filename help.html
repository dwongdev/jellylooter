‚ùì Frequently Asked Questions

Why do I get "No space left on device"?
This usually means your Docker volume isn't mapped correctly. The /storage path inside the container needs to point to your actual media storage. Check your Docker volume mappings and ensure the path is writable.
Why aren't my downloads faster even with speed limit disabled?
Speed depends on your network, the remote server's upload speed, and any limitations they may have. Also check the "Max Concurrent Downloads" setting - more simultaneous downloads can increase throughput.
How do I get an API key from Jellyfin?
In Jellyfin, go to Dashboard ‚Üí API Keys ‚Üí Add. Give it a name like "JellyLooter" and copy the key. Alternatively, you can use username/password authentication in JellyLooter.
Why isn't duplicate detection working?
Make sure you've configured your local server in Settings and clicked "Rebuild Cache". Detection works by matching IMDB/TMDB IDs, so both servers need metadata providers configured.
Can I change the number of concurrent downloads?
Yes! Go to Settings ‚Üí Advanced ‚Üí Max Downloads. Changes take effect immediately without restarting the container.
Where are downloads saved?
Downloads go to the path you select when starting a download. This path is inside /storage in the container, which should be mapped to your actual storage location.

üîß Troubleshooting

Check Docker volume mappings:
docker inspect jellylooter | grep -A10 "Mounts"
Test write permissions:
docker exec jellylooter touch /storage/test && docker exec jellylooter rm /storage/test
Check actual disk space:
docker exec jellylooter df -h /storage
View container logs:
docker logs -f jellylooter
